cEventLog :: A DSC Resource for managing Windows Event Logs
This repository contains the cEventLog PowerShell module, containing Microsoft Windows PowerShell Desired State Configuration (DSC) resources to manage Windows Event Log.

Background
This module is exactly like the xWinEventLog DSC Resource, except that it has been modified to include a Path parameter for changing the location where an Event Log is stored.  

Installation
To install the cEventLog PowerShell module, download it and unzip it to $env:ProgramFiles\WindowsPowerShell\Modules.
